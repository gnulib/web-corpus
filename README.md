# web-corpus
Web crawler to build corpus text from given domains

## Usage
```
usage: httpCrawler.py [-h] [-d DIR] [-u URL] base

positional arguments:
  base               base domain for crawling

optional arguments:
  -h, --help         show this help message and exit
  -d DIR, --dir DIR  directory name to write text corpus
  -u URL, --url URL  url to start crawling
```
